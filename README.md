# Machine Learning

A comprehensive collection of machine learning models and algorithms implemented in Jupyter notebooks, covering various techniques from basic regression to advanced neural networks.

## About

This repository contains code tested through different machine learning models, providing practical implementations and examples for learning and reference purposes. Each notebook demonstrates specific ML techniques with real-world datasets.

## Repository Contents

The repository includes implementations of the following machine learning models and techniques:

- **ANN-DNN.ipynb** - Artificial Neural Networks and Deep Neural Networks
- **Airline_customer_satisfaction.csv** - Customer satisfaction analysis dataset
- **Comparing_ML_DL_Models.ipynb** - Comparative analysis of different ML and DL approaches
- **LinearRegression.ipynb** - Linear regression modeling
- **LOGISTIC_REGRESSION.ipynb** - Logistic regression for classification tasks
- **SVM_MODEL_.ipynb** - Support Vector Machine implementations
- **SVM_DT_RF.ipynb** - Support Vector Machines, Decision Trees, and Random Forests comparison
- **SVM_and_Descison_Tree.ipynb** - Detailed SVM and Decision Tree analysis
- **Spam_Detection_NLP.ipynb** - Natural Language Processing for spam detection
- **_CNN.ipynb** - Convolutional Neural Networks
- **NLP_BASICS.ipynb** - Natural Language Processing fundamentals
